# Assignment-1
https://youtu.be/YbkCow93Ns4
Greetings, I've developed an Android App using Kotlin in Android Studio that provides social interaction suggestions based on a users input time of day. The user interface consists of a text input for the time of day which is a text field where the user can type in the time of day for example, "morning", "afternoon" or "dinner". A submit button is present to submit whatever time of day the user has input. A text view is present to display a social suggestion based on the entered time of day and a reset button is present to clear the input and the suggestion. This app is coded to cater for 6 time of day inputs namely being, "morning", "mid-morning", "afternoon", "afternoon snack time", "dinner" and "after dinner/night". If the user enters an input that is outside of these 6 time of day inputs then the app will provide constructive feedback to notify the user that they have entered invalid information.
I've used GitHub to upload the MainActivity.kt and activity_main.xml files.
I chose a light blue background and added a clock icon to fit the aesthetic of the application.
